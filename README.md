# FLASK TEMPLATE

this template provides very basic flask template with html and basic css.

# SETUP ENVIRONMENT

- 1 CREATE NEW FOLDER
- 2 cd <new folder>
- 3 Create a a Virtual Environment:

```bash
python -m venv venv
```

-4 Acticate the Virtual Environment:

```bash
\venv\Scripts\activate
```

-5 Run flask

```bash
python run.py
```
